# RTS PRO Starter Kit (Quick Start)

## macOS (recommended)
1. Download the ZIP
2. Unzip it
3. Double-click **START_HERE.command**

If macOS blocks it:
- Right click -> Open -> Open

## What you get
- A **safe baseline** you can always roll back to
- A simple guided setup (no special configuration)
- A folder structure for **logs / evolution / agent work**

## Optional: GitHub backup (easy way)
Use **GitHub Desktop**:
- Add Existing Repository -> select this folder
- Publish repository

> Nothing is uploaded unless you publish/push.
