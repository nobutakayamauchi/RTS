Why RTS?

Modern AI workflows restart thinking every session. Ideas disappear. Decisions fragment. Execution history is lost.
RTS preserves reasoning continuity across AI sessions.

Local run

RTS runs fully local. No cloud dependency. No telemetry.

Open:

starter/python

Then run:

python START_HERE.py

RTS creates:

logs/ memory/ runs/ evolution/

Execution history becomes operational memory.
Execution history survives session resets and device changes.

Platforms: Windows / Mac / Linux / Android (Termux supported).
iPhone: use a synced environment (Mac/PC) or a remote dev box; iOS cannot run full Python reliably without external tooling.
