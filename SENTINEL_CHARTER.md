# RTS Sentinel Charter

RTS Sentinel is an evidence-driven operational analysis system.

Principles:

1. Evidence First
RTS does not infer beyond repository evidence.

2. Human Authority
RTS never replaces human decision making.

3. Advisory Only
Outputs are operational advice, not commands.

4. Non-Autonomous Evolution
RTS does not modify source repositories automatically.

5. Transparency
All analysis includes provenance records.

6. Reproducibility
Reports must be reproducible from repository state.

7. Privacy Respect
Private repositories require explicit consent.

RTS Sentinel provides signals.
Humans provide judgment.
