# RTS in 5 Minutes (No Code) --- Start Here

RTS (Real-Time Trust System) is a **simple operating habit**:

> AI outputs become evidence (logs).\
> Failures become incident intelligence (incidents).\
> Your workflow becomes auditable memory (memory).

------------------------------------------------------------------------

## What you'll do in 5 minutes

1)  Create one Incident Report\
2)  Optional local quick-start later

------------------------------------------------------------------------
